## Installation

1. Set up a Node.js environment (tested with Node.js 18.15.0). It's recommended to use a virtual environment using [ASDF-VM](https://asdf-vm.com/).
2. Install dependencies: `npm install`.
3. Run! (see below)

## Usage

Start the vulnerable app:

```
node app.vuln.js
```

Start the fixed app:

```
node app.fix.js
```
