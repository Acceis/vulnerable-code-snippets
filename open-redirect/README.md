## Installation

1. Set up a Ruby environment (tested with Ruby 3.2.0). It's recommended to use a virtual environment using [ASDF-VM](https://asdf-vm.com/).
2. Install dependencies: `bundle install`.
3. Run! (see below)

## Usage

Start the vulnerable app:

```
VULN=on rackup
```

Start the fixed app:

```
VULN=off rackup
```
